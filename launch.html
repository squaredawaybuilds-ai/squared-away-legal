Launch the Squared Away QuickBooks Connector

The Connector is a local command-line application. There is no remote server to launch — it runs entirely on the operator's own device, communicating directly with Intuit's QuickBooks Online API.

To launch the Connector

  1. Open a terminal on the device where the Connector is installed.
  2. From the Connector's installation directory, run the local entry point: 
    uv run quickbooks-mcp
  3. Once running, accounting tools (reports, invoices, bills, customers, etc.) are available via the local interface that the operator uses (e.g., the Claude Code CLI, when the Connector is registered as an MCP server).

First-time use

Before the Connector can be launched against a QuickBooks Online company, OAuth authorization must be completed. See Connect for instructions.

Disconnecting

See Disconnect for full instructions on revoking authorization and removing local credentials.
