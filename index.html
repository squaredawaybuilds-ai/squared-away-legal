Squared Away QuickBooks Connector

A private, internal-use integration that connects Squared Away Builds' own QuickBooks Online (Canada) company to a local computing environment, via Intuit's official OAuth 2.0 authorization flow.

What it does

The Connector is a local command-line tool that reads accounting data (Profit & Loss, Balance Sheet, A/R and A/P aging, invoices, bills, customers, vendors) and creates or updates invoices, estimates, and bills in Squared Away Builds' authorized QuickBooks Online company. It is operated solely by Squared Away Builds for its own bookkeeping automation.

Scopes used

  • com.intuit.quickbooks.accounting — required to read accounting reports and read/create accounting transactions.

The Connector does not request the QuickBooks Payments scope and cannot move money or process payments.

How it works

The Connector is a local Python application invoked from a command line on the operator's own device. On first run, the operator triggers Intuit's OAuth 2.0 authorization flow, which issues an access token and a refresh token. Tokens are stored locally with owner-only file permissions. All subsequent API calls go directly between the operator's device and Intuit's servers over HTTPS.
